Coffee Shop

No projeto Coffee Shop, utilizamos HTML, CSS e Bootstrap para desenvolver o website desta cafeteria moderna e acolhedora. O site foi projetado para oferecer uma experiência de usuário agradável, destacando o menu de produtos e informações sobre o local. A utilização do Bootstrap permitiu uma interface responsiva, garantindo uma experiência consistente em diferentes dispositivos.

Blog Chase Atlantic

No projeto Blog Chase Atlantic, utilizamos somente HTML e CSS para criar um espaço online dedicado à banda Chase Atlantic. O blog apresenta informações sobre os integrantes da banda, trechos de letras de músicas e curiosidades sobre o grupo. Com um design simples e elegante, o site oferece uma experiência imersiva para os fãs da banda, destacando sua música e estilo distintivos.

HDC Host

Utilizando apenas HTML e CSS, desenvolvemos um site intuitivo e informativo que apresenta os diferentes planos oferecidos, recursos disponíveis e instruções para o registro de domínios. Com um design limpo e responsivo, o HDC HOST proporciona uma experiência simplificada para clientes que buscam criar e hospedar seus próprios sites.

HBO Clone

No projeto Clone do HBO, utilizamos apenas HTML e CSS para criar uma versão simulada do site da HBO. Incorporamos animações, cards e outros elementos de design para replicar a experiência de navegação oferecida pelo serviço de streaming.

Calculadora de IMC

Desenvolvemos uma Calculadora de Índice de Massa Corporal (IMC) utilizando HTML, CSS e JavaScript. Esta ferramenta interativa permite aos usuários calcular seu IMC com base em seu peso e altura.

Discord Clone

No projeto Clone da Página Principal do Discord, utilizamos apenas HTML e CSS para replicar a interface da página principal do Discord. Criamos uma versão simulada do layout, elementos visuais e estilo do Discord.

Instagram Login Clone

No projeto Clone da Página de Login do Instagram, utilizamos apenas HTML e CSS para reproduzir fielmente a interface de login da plataforma Instagram. Desenvolvemos campos de entrada para o nome de usuário e senha, além de botões interativos para acesso à conta e recuperação de senha. O layout e o estilo foram cuidadosamente replicados para proporcionar uma experiência visual semelhante à página de login oficial do Instagram.
